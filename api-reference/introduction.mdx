欢迎使用API中转站接口文档！这里提供了所有支持的AI模型的详细接口说明。

API中转站提供统一的RESTful API接口，兼容OpenAI API格式，支持多种AI服务：

## 基础信息

**API基础地址**: `http://gptproto.com`

**认证方式**: Bearer Token

**请求格式**: JSON

**响应格式**: JSON

## 快速开始
