# Create Image

: 

## 接口信息

- **请求方法**: `POST`
- **接口路径**: `/v1/images/generations`
- **认证方式**: Bearer Token
- **标签**: qwen/image/generations

## 认证说明

请在请求头中包含有效的 Bearer Token：

```bash
Authorization: Bearer YOUR_API_KEY
```
