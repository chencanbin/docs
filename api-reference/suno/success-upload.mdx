# Success Upload

:  - Suno AI音乐生成接口

## 接口信息

- **请求方法**: `POST`
- **接口路径**: `/v1/suno/uploads/audio-url`
- **认证方式**: Bearer Token
- **标签**: 'Suno/Scenario 5: Upload Custom Audio and Continue'

## 认证说明

请在请求头中包含有效的 Bearer Token：

```bash
Authorization: Bearer YOUR_API_KEY
```

1. 在右侧面板中输入请求参数

## 示例代码

### cURL 示例

```bash
curl -X POST "http://129.226.58.30/v1/suno/uploads/audio-url" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

### Python 示例

```python
import requests

url = "http://129.226.58.30/v1/suno/uploads/audio-url"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}

response = requests.post(url, headers=headers)
print(response.json())
```
