# veo3-pro

:  - Google Gemini模型接口

## 接口信息

- **请求方法**: `POST`
- **接口路径**: `/v1/veo/videos`
- **认证方式**: Bearer Token
- **标签**: Gemini/veo3 Video/Asyn mode

## 认证说明

请在请求头中包含有效的 Bearer Token：

```bash
Authorization: Bearer YOUR_API_KEY
```
